Memulai pembelajaran git bersama eduwork
==> Git sendiri merupakan sebuah Git adalah perangkat lunak pengendali versi atau proyek manajemen kode perangkat lunak yang diciptakan oleh Linus Torvalds,
    yang pada awalnya ditujukan untuk pengembangan kernel Linux. Desain Git terinspirasi oleh BitKeeper dan Monotone.

Materi yang dibahasa pada pembelajaran GIT yaitu :
  1. Intro
    ==> Pada bagian ini kita akan mengenal kegunaan,defini dan manfaat git.Serta penerapannya dalam pemrograman

  2. Fundamental
    ==> Pada bagian ini kita akan membahas perintah-perintah dasar yang digunakan dalam GIT :
            1. Git Init
            2. Git Status
            3. Git Add
            4. Git Commit
            5. Git Log
  3. Branch
    ==> Pada bagian ini kita akang mengenal percabangan pada repository
  4. GUI
    ==> Pada materi ini kita akan membahas tools yang digunakan untuk mempermudah dalam menggunakan GIT
